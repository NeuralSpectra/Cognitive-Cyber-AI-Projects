# **Project 31: Advertisement Predictor**

This project predicts whether a user will click on an advertisement based on features such as **Age, Time, Country, City, Ad Topic Line,** and other related attributes.

It demonstrates how machine learning can be applied to marketing analytics to understand user behavior and optimize ad targeting.

---

## **How to Use the Streamlit Application**

1. Ensure **Python 3.8+** is installed with an editor like **VS Code** or **PyCharm**.

2. Place the following files in the **same directory**:

   * **app.py**
   * **Finalized-Model.pickle**
   * **Scaler_X.pickle**
   * **LabelEncoders.pickle**

3. Open a terminal in that directory and run:

   * **streamlit run app.py**

4. A browser tab will open automatically, allowing you to enter user details and predict ad click probability interactively.
